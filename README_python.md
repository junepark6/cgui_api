# Python API Client & CLI for CHARMM-GUI

The Python toolkit provides a robust, extensible way to interact with the CHARMM-GUI backend API, including job submission, monitoring, job history tracking, and advanced search capabilities.

---

# Features

- `CharmmGUIClient` for communication  
- `QuickBilayer` API module  
- `charmmgui` command-line tool  
- Secure login with hidden password  
- Pretty-printed job status  
- Redis Queue info + rank  
- Job history storage:
  - JSON mode (default)
  - SQLite mode (searchable)  
- Job search:
  - text filters  
  - date filters  
  - module filters  

---

# Login

```bash
charmmgui login
```

Prompts for:
Email:
Password: (hidden)

Token saved at: ~/.charmmgui_token

# Submit Quick Bilayer Job
```bash
charmmgui bilayer \
    --jobid 1234567890 \
    --upper DOPC:POPC=1:1 \
    --lower DOPC:POPC=1:2 \
    --margin 20 \
    --run-ppm \
    --run-ffconverter \
    --clone_job
```
Supports boolean flags:
- --run-ppm
- --run-ffconverter
- --clone_job
- --prot_projection_upper
- --prot_projection_lower

# Check Job Status
```bash
charmmgui status --jobid 6539798673
charmmgui status
```
Pretty print includes:
- Job state
- rqinfo
- rank (jobs ahead)
- last output file
- last 30 log lines

When no job ID is provided, the status command prints the logged-in user's job table.

Enable/disable in config.py:
```python
PRETTY_STATUS_OUTPUT = True
```
# Job History (JSON or SQLite)
## JSON mode 
```python
STORAGE_BACKEND = "json"
```
Job records saved to:
```bash
~/.charmmgui/jobs/<jobid>.json
```

## SQLite mode (default)
```python
STORAGE_BACKEND = "sqlite"
```
Then:
```bash
charmmgui jobs list
charmmgui jobs info --jobid 6539798673
charmmgui jobs search --contains POPC
charmmgui jobs delete --jobid 6539798673
charmmgui jobs clean
charmmgui jobs clean --days 14
```

`jobs clean` removes SQLite job-history records older than 7 days by default,
matching CHARMM-GUI's approximate remote job-folder retention window.

## Search Jobs

Examples: 
Search by lipids:
```bash
charmmgui jobs search --contains POPC
```
By module:
```bash
charmmgui jobs search --module quick_bilayer
```

By date:
```bash
charmmgui jobs search --before 2025-12-10
```
Combine filters:
```bash
charmmgui jobs search --contains POPC --after 2025-12-01 --module quick_bilayer
```

## Python API Usage

```python
from charmmgui import CharmmGUIClient, QuickBilayer

client = CharmmGUIClient(token=CharmmGUIClient.load_token())
qb = QuickBilayer(client)

jobid = qb.submit(
    jobid="1234567890",
    upper="DOPC:POPC=1:1",
    lower="DOPC:POPC=1:2",
    margin=20
)
print(jobid)
```

`CharmmGUIClient(token=...)` expects an API token value. The safest pattern is
to call `CharmmGUIClient.load_token()`, which reads the saved token from
`session.token` in the current directory or `~/.charmmgui_token`.
This token is generated by running `charmmgui login`, then entering your user
email account and password at the prompts.

If your installed version supports token path resolution, `token` may also be a
path to a token file:

```python
client = CharmmGUIClient(token="/Users/sangjun/.charmmgui_token")
```

For older installed versions, do not pass the token file path directly. Passing
the path as the token value can cause an API error such as:

```text
{"error":"Invalid token","detail":"Wrong number of segments"}
```

# Configuration (config.py)
```python
STORAGE_BACKEND = "json"      # "json", "sqlite", or "none"
JOB_RECORD_DIR = "~/.charmmgui/jobs"
SQLITE_DB_FILE = "~/.charmmgui/job_history.db"

PRETTY_STATUS_OUTPUT = True
QUIET_DEFAULT = False
VERBOSE_DEFAULT = False
```
