# Bash Scripts for CHARMM-GUI API Access

This directory provides minimal Bash-based tools for interacting with CHARMM-GUI API endpoints.  
These scripts are ideal for lightweight workflows, shell pipelines, automated jobs, or remote HPC environments.

---

# Available Scripts

### 1. `login.sh`
Authenticate and retrieve a JWT token.

```bash
./login.sh --email my@email.com
```

- Password is hidden
- Token is printed and saved locally if desired
- Can be used in automation pipelines

### 2. `status.sh`

```bash
./status.sh --jobid 6539798673
```

- Job status (submitted, running, done, error)
- Redis Queue information (rqinfo)
- Number of jobs ahead (rank)
- Name of last output file
- Last 30 lines of CHARMM-GUI log

### 3. `quick_bilayer.sh`
./quick_bilayer.sh \
    --jobid 1234567890 \
    --upper DOPC:POPC:CHL1=1:1:2 \
    --lower DOPC:POPC:CHL1=1:2:1 \
    --margin 20 \
    --run-ppm \
    --run-ffconverter

- protein systems
- membrane-only jobs
- lipid composition strings
- membrane-type presets
- PPM orientation
- ffconverter generation
- clone-job mode

## Endpoint
[POST](https://charmm-gui.org/api/quick_bilayer) `https://charmm-gui.org/api/quick_bilayer`

## Required Parameters (Conditional)

| Parameter | Required When | Description |
|-----------|---------------|------------|
| `jobid` | Protein-embedded systems | Job ID generated by the CHARMM-GUI PDB Reader module. Not required for membrane-only systems. |
| `membrane_only=true` | Membrane-only systems | Generates a system without a protein. When used, `jobid` is not required. |
| `upper` + `lower` | Using custom lipid compositions | Lipid composition strings for upper and lower leaflets (e.g., `DOPC:POPC:CHL1=1:1:2`). |
| `membtype` | Using predefined membranes | Select one of the 18 predefined biomembrane types. |
| `margin` | Always required | XY box boundary margin. |

> Lipid composition must be specified **either** by providing `upper` and `lower` strings **or** by selecting a predefined `membtype`.

---

## Optional Parameters

| Parameter | Default | Description |
|-----------|---------|------------|
| `wdist` | 22.5 | Z-length water boundary |
| `Ion_conc` | 0.15 | Ion concentration (M) |
| `Ion_type` | NaCl | Ion type |
| `clone_job` | false | Copy existing job directory before execution |
| `ppm` | false | Enable PPM alignment |
| `prot_projection_upper` | false | Project protein atoms to upper leaflet |
| `prot_projection_lower` | false | Project protein atoms to lower leaflet |

### Example: user-specified lipid composition strings 

```bash
bash cli_quick_bilayer.sh --jobid 1234567890 \
  --upper POPC:CHL1=3:2 --lower POPC:CHL1=3:2 \
  --margin 20.0 --ion_type KCl
```

### Example: preset membrane types

```bash
bash cli_quick_bilayer.sh --membrane_only \
  --membtype PMm --margin 30.0
  --ion_type KCl --wdist 20.0
```



### 4. `download.sh`

```bash
./download.sh --jobid 6539798673 -o results.tgz
```

# Requirements
- curl
- jq
- bash 4+

# Notes
- Bash scripts do not implement job history tracking
- For long-term workflows, consider using the Python toolkit (see README_python.md)
- All Bash tools rely on the CHARMM-GUI API backend
